# Markdown结构规范

## 完整结构模板

```
## 方剂名称
**方剂名称 · 功能描述的"方剂类型"方**
- 方剂分类（可选）

**[药物手绘组合：药材1、药材2、药材3]**

**核心口诀**
"口诀内容"

主治：**病机描述**：
1.  **主要症状1**：详细描述
2.  **主要症状2**：详细描述
- 兼证症状（可选）

**病机剧场 · 场景标题**

[场景描述：生动的比喻和场景]

**药物行动队**

- **组名1**：药材，作用描述。如同**类比描述**。
- **组名2**：药材，作用描述。如同**类比描述**。

**症状-药物连线**

- 症状1 ← 药材 (作用)
- 症状2 ← 药材 (作用)

**临床超级提醒**
- **场景**：**适用疾病**属相关证型者。
- **用法**：用法说明（可选）

**经典原文**

> "经典原文内容"
> —— 出处
```

## 必选章节说明

### 1. 方剂名称
- 格式：`## 方剂名称`
- 示例：`## 理中丸`
- 要求：使用标准方剂名称

### 2. 副标题
- 格式：`**方剂名称 · 功能描述的"方剂类型"方**`
- 示例：`**理中汤 · 中焦虚寒的"暖炉"方**`
- 可选分类：`- 温中袪寒剂`

### 3. 药物手绘组合
- 格式：`**[药物手绘组合：药材1、药材2]**`
- 示例：`**[药物手绘组合：干姜、人参、白术、炙甘草]**`
- 要求：列出所有组成药材，用顿号分隔

### 4. 核心口诀
- 格式：
  ```
  **核心口诀**
  "口诀内容"
  ```
- 示例：`"理中汤主理中乡，甘草人参术干姜；呕利腹痛阴寒盛，温中祛寒补脾阳。"`
- 要求：使用经典方歌或创作易记口诀

### 5. 主治
- 格式：
  ```
  主治：**病机描述**：
  1.  **主要症状1**：详细描述
  2.  **主要症状2**：详细描述
  - 兼证症状（可选）
  ```
- 示例：`主治：**脾胃虚寒，中焦不运**：`
- 要求：病机描述准确，症状详细

### 6. 病机剧场
- 格式：
  ```
  **病机剧场 · 场景标题**
  
  [场景描述：生动的比喻]
  ```
- 示例：`**病机剧场 · 寒冬里的老旧锅炉房**`
- 要求：使用生动比喻（锅炉房、河流、农田等）

### 7. 药物行动队
- 格式：
  ```
  **药物行动队**
  
  - **组名**：药材，作用。如同**类比**。
  ```
- 示例：`- **点火升温组**：干姜，辛热，温中散寒。如同**加大火力**。`
- 要求：分组合理，类比生动

### 8. 症状-药物连线
- 格式：
  ```
  **症状-药物连线**
  
  - 症状 ← 药材 (作用)
  ```
- 示例：`- 脘腹冷痛 ← 干姜 (温中散寒)`
- 要求：明确对应关系

### 9. 临床超级提醒
- 格式：
  ```
  **临床超级提醒**
  - **场景**：**疾病**属证型者。
  - **用法**：说明（可选）
  ```
- 示例：`- **场景**：**慢性胃炎、消化性溃疡**属中焦虚寒者。`

### 10. 经典原文
- 格式：
  ```
  **经典原文**
  
  > "原文"
  > —— 出处
  ```
- 示例：`> "大病差后，喜唾，久不了了，胸上有寒，当以丸药温之，宜理中丸。" —— 张仲景《伤寒论》`

## 可选章节

### 1. 使用铁律
```
**使用铁律**

✅ **核心铁诊（必须同时具备）**：
- 症状：描述
- 症状：描述

**兼见证**：症状。
```
**使用条件**：方剂有明确的诊断标准时使用（如炙甘草汤）

### 2. 病机演变
```
### **病机演变**

病因 → 病机变化1 → 病机变化2 → 最终症状
```
**使用条件**：病机比较复杂的方剂需要增加（如黄芪桂枝五物汤）

### 3. 配伍点睛
```
**配伍点睛**：**关键词**，简要说明。
```
**使用条件**：有独特配伍、有代表性意义的方剂需要增加

### 4. 方剂对比
```
**方剂对比**

**方剂A vs 方剂B**
- **相同点**：简要说明
- **不同点**：
  1. **病机**：方剂A的病机 vs 方剂B的病机
  2. **症状**：方剂A的症状特点 vs 方剂B的症状特点
  3. **治法**：方剂A的治法 vs 方剂B的治法
  4. **药物**：方剂A的核心药物 vs 方剂B的核心药物
```
**使用条件**：包含两类情况：
1. **同病异治**：相同症状不同病机的方剂对比
2. **重要加减方对比**：基础类方剂的重要加减方（如四逆汤系列、平胃散加减等）

### 5. 兼证加减
```
**兼证加减**：
- 兼证1 → 加药1（新方名）
- 兼证2 → 加药2（新方名）
```
**使用条件**：方剂有重要的临床加减变化时使用

### 6. 用法要点
```
- **用法要点**：特殊用法说明
```
**使用条件**：方剂有特殊用法或注意事项时使用

## 创作要点

### 核心原则
1. **经典性优先**：卡片空间有限，只放最经典、最有代表性的内容
2. **生动拟人化**：药物行动队要拟人化展示，与病机剧场形成联动
3. **特色突出**：每个方剂要突出其独特之处，避免泛泛而谈

### 内容选择标准
- ✅ **必须包含**：经典配伍、核心病机、代表性用法（如麻黄汤喝后要和热粥）
- ✅ **推荐包含**：生动比喻、拟人化描述、特色配伍
- ❌ **避免包含**：泛泛的使用要点、普通临床应用、缺乏特色的注意事项

### 药物行动队优化要求
1. **拟人化角色**：为每个药物组创建拟人化角色（如"点火升温组"的干姜是"锅炉工"）
2. **与病机剧场联动**：药物行动队的角色要在病机剧场场景中发挥作用
3. **具体配伍展示**：明确展示药物之间的配伍关系和协同作用

### 示例对比
- **差**："使用要点：水煎服，每日1剂"（无特色）
- **好**："用法：服后需啜热粥一碗，以助药力"（有特色，如麻黄汤）
- **差**："临床应用：慢性胃炎"（泛泛）
- **好**："适用：慢性胃炎属中焦虚寒者，见脘腹冷痛、喜温喜按"（具体）

- [ ] 是否避免了泛泛的使用要点？
- [ ] 是否突出了该方剂的独特之处？
